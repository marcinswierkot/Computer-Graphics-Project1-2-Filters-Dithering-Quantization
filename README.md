# Image processing
WPF image editor with the following selected filters.
## Features

- Function filters: 
    - Inversion 
    - Brightness correction 
    - Contrast enhancement 
    - Gamma correction
- Convolution filters:
    - Box blur
    - Gaussian blur
    - Sharpen
    - Emboss
    - Edge detection
- Grayscale
- Color Quantization: 
    - Median Cut algorithm
- Average Dithering:
    - k graylevels for grayscale
    - R G B values for color dithering
